# UI Design Document: GoVip

## 1. Design Philosophy
- **Style**: Modern, clean, and minimalist.
- **Color Palette**: Neutral background (Zinc/Slate), Primary accent (Blue/Indigo) for actions.
- **Typography**: Sans-serif (Inter or system default) for readability.

## 2. Key Views
### Home Page (Event List)
- **Header**: Logo and Navigation.
- **Hero Section**: Brief welcome message.
- **Grid Layout**: Cards displaying events. Each card includes:
  - Event Image
  - Title (Bold)
  - Date & Location (Muted text)
  - Price (Highlighted)

## 3. Components
- **EventCard**: Reusable component for displaying event summary.
- **Button**: Standardized primary/secondary buttons.
- **Layout**: Container with consistent padding and max-width.
